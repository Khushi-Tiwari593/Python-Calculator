# Python Calculator
A simple command-line calculator built using Python. This project performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* User-friendly input and output
* Beginner-friendly Python project

## Technologies Used
* Python 3

## Sample Output
Enter first number: 10
Enter second number: 5
Choose operation (+, -, *, /): +
Result: 15
